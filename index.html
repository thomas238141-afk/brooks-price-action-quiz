复习:四步操作流程(判断框架)

第一部分 · 综合判图(10 题)

判:状态 / Always-In 方向 / 打法 / 进场(stop 还是 limit) / 止损。

第二部分 · 区间挂单(5 题)

写全套下单:方向 / limit 挂哪几价 / 止损 / 目标 / 分几批。⚠️ 其中两题是"趋势伪装成区间"的陷阱。

附:十条核心铁律(自用提炼)

  1. context 先于 direction。先判趋势/区间、主要趋势线破没破,再决定数 H(买)还是数 L(卖)。
  2. 破主要趋势线之前,绝不逆势。这是全书最重要的一条;反转必先破线,再回测极限点。
  3. 状态决定进场模型。趋势用 stop 单顺势追强;区间用 limit 单逆势接弱。别在区间里用趋势思维追动能。
  4. EMA 走平 = 区间头号信号。价格在两水平线间折返、边缘出反转棒→fade 边缘,别追突破。
  5. 多数突破会失败。别裸追突破棒;买突破回调(回测突破位=失败再失败=高胜算)。
  6. 下跌里反弹到走平/下行 EMA = 卖点(M2S),不是买点。上涨镜像同理。
  7. 两次失败 → 反向。市场尝试做一件事两次都没成,通常会做相反的事。
  8. 高潮棒是止盈/反手信号,不是进场信号。巨幅+长上/下影出现在趋势极端处,别追。
  9. 先判"是不是区间"。EMA 斜率+高低点结构是铁证;把趋势当区间去 fade,方向系统性反。
  10. 读全部,做少数。每天只做 2–5 个最好的结构;稳定盈利后扩大仓位,而不是增加打法。

设置 AI 老师

填入你自己的 API,AI 会按标准答案给你逐项点评打分。密钥只存本机浏览器(localStorage),不上传。

仅保存在本机浏览器,不发送到任何第三方服务器(除你选的服务商本身)。
部分服务商(如 DeepSeek/Kimi)可能不允许浏览器直连(CORS)。若测试失败,请下载项目后用 node server.js 本地运行,内置代理可支持任意服务商。Claude 与 OpenAI 一般可浏览器直连。
